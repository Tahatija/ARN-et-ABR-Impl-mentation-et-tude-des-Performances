# Arbres Rouge-Noir et ABR — Implémentation et Analyse Java

## Description
Projet académique Java implémentant des **arbres rouge-noir (ARN)** et des **arbres binaires de recherche classiques (ABR)**.  
Le projet inclut une **implémentation complète**, des **tests unitaires** et une **étude expérimentale** comparant les performances des deux structures pour différentes tailles et ordres d’insertion.  

L’objectif est de comprendre le fonctionnement des ARN, valider leur équilibrage automatique, et analyser leurs avantages par rapport aux ABR classiques.

---

## Fonctionnalités
- Insertion, suppression et recherche dans les ABR et ARN  
- Rééquilibrage automatique des ARN pour garantir une complexité logarithmique  
- Parcours des arbres : infixe, préfixe et postfixe  
- Tests unitaires pour toutes les méthodes  
- Étude expérimentale des performances pour différentes tailles d’arbre et ordres d’insertion
